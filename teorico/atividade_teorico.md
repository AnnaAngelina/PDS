# Atividade

a) O que é software?

Software é um conjunto de dados, informações (transformadas e controladas) e ferramentas para que os programas operem corretamante, mantendo também as funções vitais e atualizações de acordo com as necessidades do usuário e ambiente. 
Sendo assim, ele é constituído por instruções, estruturas de dados e documentação que oferecem uma base para o controle do computador e comunição entre seus componentes, além de outros recursos conjuntos, como interface e hardware.
Eles podem ser genéricos, quando desenvolvidos para qualquer pessoa (baseado em necessidades do mercado), ou sob encomenda, desenvolvidos sob especificações do cliente. 

b) Cite três diferenças entre hardware e software.

O software pode ser representado pelo hardware, pois ele é lógico, enquanto o hardware é físico. 
Nesse sentido, o software não é fabricado como o hardware, mas desenvolvido, assim como não pode ser desgastado, o que característicamente é atributo de hardwares que, com o tempo, têm seus componentes deteriorados e, geralmente, inutilizáveis.

c) O que é engenharia de software?

Engenharia de software abrange a utilização de princípios os quais possuem foco na qualidade, processos e métodos, tendo, além disso, um leque das ferramentas que ajudam no suporte automatizado (CASE) para garantir a construção, das concepções iniciais até a morte, de softwares de altíssima qualidade.

d) Para você, quais desafios e facilidades o futuro trará para
o desenvolvimento de software?

Como a tecnologia avança de modo muito rápido, no futuro, há a possibilidade de que essa área seja amplamente beneficada, com ferramentais mais sofisticadas, como observamos, inclusive atualmente, com o desenvolvimento das IAs. 
Talvez as dificuldades se encontrem justamente nas próprias consequencias de criar uma tecnologia tão especializada e abrangente. A questão é: será que saberíamos lidar com isso com responsabilidade? Muitos problemas surgem disso e dificilmente podemos prever tão bem nossas proprias reações
