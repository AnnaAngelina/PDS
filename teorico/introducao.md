# Introdução

## O que é software?

Programas do computador, dados, instruções, estrutura de dados para que o programa opere corretamente. 

- Genéricos (prateleira): desenvolvidos para qualquer pessoa. Maseado em necessidades de mercado.

    ex: Office da Microsoft

- Sob encomenda (personalizados): Desenvolvidos de acordo com as especificações do cliente

### A natureza do software

Produto que fornece o potencial computacional representado pelo hardware. Ele é um **transformador de informações**. É um elemento lógico, por isso, passa a ser desenvolvido 

Informações --> Dados pessoais em informações úteis

Ele pode se tornar defasado

- Facilidade de manutenção
- Confiança 
- Eficiência
- Usabilidade

### Evolução 

Softwares eram muito pequenos, visto as limitações de hardware. Com o crescimento do poder computacional, desenvolve-se maior complexidade

### Software legado

Dão suporte a funções vitais e indispensáveis. São antigos, mas evoluem com o tempo de acordo com as necessudades de novos ambientes, requisitos de negócios etc.

São dispendiosos de manter e arriscados de evoluir

Enfrentou crise por estimativas de prazo, insatisfação do cliente, dificuldade de manter.

*A engenharia de software é importante pois nos capacita para o desenvolvimento de sistemas complexos*
